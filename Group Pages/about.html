Christan

First year CS student

As a student, I've only just started into Computer Science after leaving medicine for less patient-centered pastures. Unlike other people my age, I never used MySpace/Tumblr/Neopets, so I haven't played around with HTML before. That said I've been having a blast with it. I've started making basic web-based JS pages to streamline my fiancee's auditing work, all using what we've learned as a springboard. I'm excited to see what we learn in the second part of this semester.

My personal pages are the Halifax, Rochester, History and Bracket Creator pages. I made the layout and first pass for the current page and the same basic layout is reused for the the References page. I also made a flexible and modular layout for the Catalog page. The slideshow on the index page was also built by me using photos uploaded by the team.
